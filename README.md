(1) What was the problem you were solving in the projects for this course?
The focus of the projects in this course was to parse file input into different data structures (hash table, linked list, binary tree) 
and analyze the efficiency of each data structure in storing and manipulating data.
    
(2) How did you approach the problem? Consider why data structures are important to understand.
Memory management and performance are important to consider when choosing a data structure to solve a problem. In this course,
I built a program with each data structure and analyzed the performance of each. I used the results of this analysis to choose
a structure and build out the problem using my understanding of the chosen structure and the requirements defined for the program. 
    
(3) How did you overcome any roadblocks you encountered while going through the activities or project?
Most of the roadblocks I encountered had to do with creating keys for hash tables and managing references. These problems were easily 
solved with some online research and studying code written by other developers using the same data structure. 

(4) How has your work on this project expanded your approach to designing software and developing programs?
This course has expanded my knowledge of data structures, memory management and performance analysis. In the 
future I can use the knowledge gained in this course to create better performing programs and identify and prevent 
memory leaks. 

(5) How has your work on this project evolved the way you write programs that are maintainable, readable, and adaptable?
Standard code-writing practices were required in this course, which included documenting code, using and understanding
data structures and analyzing the performance of a program. I intend on using these lessons in the future to write
higher-quality and maintainable programs.
